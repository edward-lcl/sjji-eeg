## Rules that shape the writing (from the CFP)
- **8 pages** of content max, **+ up to 2 pages** for references.
- **Double-blind / anonymized:** no author names or affiliations in the PDF, and **do not** put
  the public repo link or anything identifying in the submitted paper (use an anonymized repo if
  you must cite code).
- **Prospects of Application:** a required paragraph at the end of the last section (Discussion or
  Conclusion) — **bold heading, max 60 words** — on the real-world application of the work. A draft
  is in `main.tex`; tighten it.
- New work only; submit via **OpenReview** (link on the CFP).
- (Track 2 is a 1-page abstract — only if we ever want the lighter option.)
