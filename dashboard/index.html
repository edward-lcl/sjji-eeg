Site Confounds, Calibration, and the Limits of Self-Supervision

The reported cross-dataset accuracy for EEG Parkinson's detection is largely a site artifact. Honest cross-site evaluation shows the real story is calibration, not signal — and self-supervised pretraining doesn't help. One page for the whole team.
Experiments: complete (seeded) Paper: outline + related-works seeded AWS GPU quota: pending

The bottom line — three findings

1 · The headline is confounded

A model using zero EEG — just "which dataset is this?" — scores 0.927 on the same pooled protocol, ≥ published models. Pooled accuracy ≠ Parkinson's detection.

2 · The "failure" is calibration

Under leave-one-site-out, supervised PD ranking still transfers (AUC 0.76). A deployable threshold recovers 0.64 balanced accuracy — the collapse was a threshold artifact.

3 · SSL doesn't help

Self-supervised pretraining (incl. disjoint TUH) and augmentation give no cross-site lift — confirmed across linear-probe, fine-tune, and data-efficiency. A clean negative.

Finding 1 — A no-EEG model beats the published numbers

Combined-protocol balanced accuracy site-confounded

Segment-level balanced accuracy on the standard pooled 4-dataset N-LNSO protocol. The site-prior null predicts each dataset's majority class using no EEG — and ds004148 (100% HC) is 66% of segments, so site ≈ label.

Why it happens

  • 4 datasets pooled; subjects held out but sites are not — the site→label shortcut stays available in every fold.
  • EEG site is trivially identifiable (amplifier, montage, filter chain).
  • The model can score high by recognizing the dataset, not the pathology.
  • Fix: report the site-prior null next to every pooled number; evaluate leave-one-dataset-out.

Finding 2 — Honest cross-site (LODO): supervised transfer is real, but calibration-bound

Dashed line = chance (0.50). Fixed-0.5 collapses (sens≈1/spec≈0) on unseen sites; a threshold chosen on the training sites (train-transferred, fully deployable) recovers it; prevalence-aware does more. ROC-AUC = 0.763 ± 0.034 (threshold-independent — proves the ranking signal is real).

Method & data notes

Protocols & metrics

  • Combined N-LNSO — the field's pooled protocol (site-confounded; report the null beside it).
  • LODO — train on 3 sites, test on the unseen 4th. The honest cross-site test.
  • Site-prior null — no-EEG dataset-majority baseline: 0.927 seg / 0.654 subj.
  • Metrics: subject-level balanced accuracy (median±IQR), ROC-AUC, bootstrap CIs; calibration policies.

Channel harmonization our finding

  • The OpenNeuro-derived 29-ch set leaves 14/29 channels dead in TUH (clinical 10-20 montage lacks them).
  • 4 recovered via old→new naming (T3→T7, T4→T8, T5→P7, T6→P8); 10 extended-10-10 positions genuinely absent.
  • We use a 19-ch TUH∩OpenNeuro montage — verified 19/19 alive — so no encoder filter trains on zeros.
